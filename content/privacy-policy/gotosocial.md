## Определение

Это независимый сервер [GoToSocial]
(с [модификациями](https://git.dc09.xyz/DarkCat09/gotosocial)), размещённый по
адресу [posts.kooltechtricks.org]. Общается по открытому протоколу
[ActivityPub], который позволяет подписываться на профили, просматривать посты
и взаимодействовать с ними через [Mastodon] и другие совместимые серверы
(общее название — [Fediverse]).

[GoToSocial]: https://gotosocial.org
[posts.kooltechtricks.org]: https://posts.kooltechtricks.org
[ActivityPub]: https://activitypub.rocks
[Mastodon]: https://joinmastodon.org
[Fediverse]: https://fediverse.info

## Обмен данными

Этот сервер обменивается данными с другими по открытому протоколу [ActivityPub].
Он получает **публично доступное** содержимое с внешних серверов: аккаунты,
посты, медиа и прочую информацию.

## Содержимое внешних серверов

- Медиа с внешних серверов по мере необходимости загружаются на этот сервер и
удаляются через неделю.

- Ответы (комментарии) под постами аккаунтов этого сервера доступны для
просмотра по ссылке. Это включает текст, медиа, аватары и эмодзи. Медиа
кэшируются согласно предыдущему пункту.

- Ссылки на посты аккаунтов этого сервера доступны публично.

- Сервер постоянно получает посты с известных ему внешних серверов, но они
недоступны по ссылке, пока не был оставлен ответ с этого сервера.

- Сервер получает посты согласно настройкам видимости. Устанавливайте
соответствующие настройки видимости (тихий публичный/только для подписчиков),
если хотите ограничить круг лиц, просматривающих ваши посты.

- Сервер использует [релей](https://relay.mastodon.ml), чтобы получать и
отправлять посты для большей видимости.

## Журнал (лог)

GoToSocial записывает все события, что может быть полезно для отладки. События
содержат следующие данные:

- **Время события**.
- **User-Agent:** Из какого клиента был произведён запрос.
- **Адрес сервера:** Какой внешний сервер вызвал событие.
- **Метод HTTP, URI, код состояния:** Какое действие произошло.

В текущей конфигурации отключена запись IP-адресов клиентов.
